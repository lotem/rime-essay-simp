# 简化字八股文

配方： ℞ **essay-simp**

[Rime](https://rime.im) 词汇表的简化字版本，可用于简化字为主的输入方案。

需要算法库版本 `librime>=1.5`。

## 用法

在韵书（`*.dict.yaml`）的YAML配置部分定义：

```yaml
vocabulary: essay-zh-hans
```

可以省略 `use_preset_vocabulary: true` 配置项。


## 安装

[东风破](https://github.com/rime/plum) 安装口令： `bash rime-install essay-simp`
